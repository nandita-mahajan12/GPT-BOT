AI Chat App (Streamlit + GPT-2)

This is a simple AI-based chat application built using **Streamlit** and **Hugging Face Transformers (GPT-2)**.  
Users can enter a prompt, and the app generates AI-based text responses.

## Features

- Simple and clean user interface
- Text generation using GPT-2 model
- Chat history support
- Built with Python and Streamlit

## Technologies Used

- Python
- Streamlit
- Transformers (Hugging Face)
- PyTorch
- GPT-2 Language Model
